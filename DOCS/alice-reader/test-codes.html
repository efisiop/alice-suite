Test Verification Codes

The following verification codes can be used for testing the Alice Reader application. These codes are pre-configured in the test database.

Note: These codes are for testing purposes only. In a production environment, each book would have a unique verification code.

Available Test Codes

Code Description Status
ALICE123 Primary test code for Alice in Wonderland Active
WONDERLAND Secondary test code Active
RABBIT Test code with special features Active
TEAPARTY Test code for advanced features Active
CHESHIRE Test code for teacher accounts Active

How to Use

  1. Register a new account or log in to an existing account
  2. Navigate to the verification page
  3. Enter one of the test codes above
  4. Complete the verification process

After successful verification, you will be redirected to the reader dashboard where you can access the full content of Alice in Wonderland.
