Alice Reader App

Interactive Reader Journey Simulation

Overview

The Alice Reader App is an interactive reading companion for "Alice in Wonderland" that provides tiered assistance features:

  • Word highlighting and definition lookup
  • AI Assistant for deeper understanding
  • Consultant support for personalized help

This demonstration showcases the key components of the app, including the landing page, authentication flow, reader interface, and administrative features.

Next Steps

To run the app locally, you would need to:

  1. Clone the repository
  2. Install dependencies with npm install
  3. Set up environment variables for Supabase
  4. Run the development server with npm run dev

The app is currently experiencing some issues with the development server, but the components have been successfully implemented and can be viewed individually.

Component Documentation

The following components have been implemented:

  • Landing Page: Introduces the app and its features
  • Authentication Pages: Login, Registration, Verification
  • Reader Dashboard: Personalized reading overview
  • Reader Interface: Reading experience with assistance features
  • Statistics Page: Detailed reading statistics
  • Consultant Pages: Tools for reading consultants
  • Admin Dashboard: System monitoring and management
